# ColourArea

**ColourArea** is an interactive, multi-page web design tool built to help developers and beginner designers rapidly explore and prototype color combinations on realistic website layouts. Instead of building mockups from scratch, users can experiment with different UI elements instantly.

## 🚀 Project Features
- **5 Distinct Layout Options**: Includes templates for Business Corporate, Creative Portfolio, E-Commerce Shop, Modern Blog, and a SaaS Landing Page.
- **Live Color Prototyping**: Click any layout element (headers, cards, buttons) and change its background color dynamically via a sidebar color picker.
- **Palette Export generation**: Copy all your modified colors and their associate element names as plain text HEX codes straight to your clipboard.
- **Color Reset**: A quick one-click undo button to restore all layout elements to their default states.
- **Clean Aesthetic**: Modern dark-mode tool interface paired with bright, crisp layout canvases for optimal contrast testing.

## 🛠 Technologies Used
- **HTML5**: Semantic document structure across 6 distinct pages.
- **CSS3**: Vanilla CSS utilizing CSS variables, Flexbox, Grid, and a centralized design system in `style.css`.
- **JavaScript (ES6)**: Vanilla JS manipulating the DOM, handling Event Listeners, and utilizing `Map` objects to store state combinations without any external frameworks.

## 💻 Setup Instructions
Because this is built using vanilla web technologies, there are no dependencies or local server installations required:
1. Clone this repository or download the ZIP file.
2. Unzip the folder to your local machine.
3. Simply double-click on `index.html` to open it in your default web browser (Chrome, Firefox, Safari, etc.).

## 🎨 Usage
1. Open the **Main Page** by launching `index.html`.
2. Scroll to the "Layout Templates" section and select a layout that fits your aesthetic criteria.
3. On the Layout Page, click any "colourable" element within the mock website (indicated by a blue dashed outline on hover).
4. Use the **Color Picker** in the left-hand sidebar to apply a new background color to the active element.
5. Click **Copy HEX** to export your new palette, or **Reset Colors** to start over.
